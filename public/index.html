Cdn API

Overview

CDN Service is a high-performance, cloud-native content delivery solution built with Go. This service combines modern architectural patterns with enterprise-grade features to offer:

Key Features

  • Multi-cloud storage support with MinIO and AWS S3
  • Real-time image resizing and optimization
  • Redis caching and worker pool support
  • WebSocket-based live monitoring and Prometheus metrics
  • Circuit breaker pattern and rate limiting

API Usage

Image Upload: Send a POST request to http://localhost:9090/upload
Required form-data keys: bucket, path, and file
Authorization header is required.

Image Retrieval:
Original size: http://localhost:9090/bucket-name/object-name
Resize with width: http://localhost:9090/bucket-name/w:300/image.jpg
Resize with height: http://localhost:9090/bucket-name/h:200/image.jpg
Resize with both: http://localhost:9090/bucket-name/w:300/h:200/image.jpg

Useful Links:

  • API Documentation (Swagger)
  • WebSocket Monitoring: ws://localhost:9090/ws
  • System Monitor: http://localhost:9090/monitor

Note: An authorization token is required for image upload and delete operations.
